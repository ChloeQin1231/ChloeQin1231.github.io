    • Enhanced channel interactivity using gesture-responsive scrolling and fade-in/fade-out effects

    • Developed a realtor clone using React.js 18, Firebase 9, Tailwind CSS 3, and React Router 6, featuring dynamic listings and responsive design
    • Integrated Firebase for user authentication, supporting username/password, and Google oAuth with secure password reset functionality
    • Utilized Leaflet for map displays, Swiper.js for image sliders, and designed reusable components for efficiency

  • Backend System for a Car Website :
    • Developed a robust back-end system using Spring Boot, comprising of a Vehicles REST API for CRUD operations on vehicle data, Pricing Service for vehicle pricing retrieval, and a Location API to fetch vehicle locations
    • Converted the Pricing Service into a microservice registered on a Eureka server, enhancing the service modularity and scaling capabilities
    • Implemented HTTP clients within the Vehicles API for seamless integration with the Pricing Service, facilitating real-time price retrieval for any vehicle in the inventory
    • Incorporated the use of Swagger for efficient API documentation, promoting ease of use, understanding, and testing of the developed APIs
